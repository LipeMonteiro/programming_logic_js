# 🚀 JavaScript Logic Training

Repositório com **40 exercícios progressivos de lógica em JavaScript**, organizados em 4 níveis de dificuldade.

O objetivo é fortalecer a base lógica e consolidar fundamentos essenciais para desenvolvimento front-end e back-end.

---

## 📚 Conteúdos Trabalhados

* Variáveis
* Condicionais
* Estruturas de repetição (loops)
* Arrays
* Objetos
* Funções
* Métodos Math
* Datas

---

## ⚠️ Regra Principal

> Resolva os exercícios **sem consultar respostas prontas**.
> Se travar, descreva o passo a passo da lógica antes de começar a codar.
